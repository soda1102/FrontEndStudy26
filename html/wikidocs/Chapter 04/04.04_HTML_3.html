JavaScript 변수 선언 방법

작성일: 2024년 3월 15일

var, let, const의 차이점

JavaScript에서 변수를 선언하는 방법은 세 가지가 있습니다.

1. var 키워드

var name = "김개발";
    console.log(name); // 김개발

2. let 키워드

let age = 25;
    age = 26; // 재할당 가능
    console.log(age); // 26

3. const 키워드

const PI = 3.14159;
    // PI = 3.14; // 오류! 재할당 불가능

언제 어떤 키워드를 사용할까?

  • const: 값이 변하지 않을 때 (기본 선택)
  • let: 값이 변할 수 있을 때
  • var: 특별한 경우가 아니면 사용 지양
